# Event Examples

## Purpose

This file turns the reusable weekly-event architecture into concrete mock content cycles for **Gnome Survivor**. It is not the final balance sheet. It is a content-design reference showing how **Lucky Draw Week**, **Treasure Week**, **Echo Week**, and **Festival Ledger** can feel different across seasons while still using the same repeatable framework.

This document assumes:
- the naming in `master_glossary.md`
- the structure in `event_content_templates.md`
- a long live-service arc where recurring content can grow well past 100 named roaming NPCs, reward branches, and side hooks over time

---

# Global Example Rules

## Rule 1 — one cycle should feel complete
A full 3-week cycle should always give the player:
- one short-burst excitement week
- one planning / targeting week
- one ritual / accumulation week
- one meta payoff through **Festival Ledger**

## Rule 2 — every week should touch story lightly
Each weekly example below includes at least one of:
- a **Burrow Post** chain
- a **Wanderer** group
- a **Runaway** rumor
- a **Smoke Whisper** variant
- a **Buried Clue** fragment

## Rule 3 — old content should remain partially relevant
Weeks should not fully replace prior weeks. Instead they should add:
- new reward pools
- new side-loop variants
- new Wanderers / Runaways
- new clue fragments
- new rotating shop targets

# Cycle Example 01 — The First Public Season

## Theme
The Burrow is beginning to reconnect with the outside world. Strange traffic increases. Local trade loosens. Old records surface. The first season should feel approachable, slightly chaotic, and optimistic on the surface.

## Festival Ledger title
**Festival Ledger: Open Paths**

## Festival Ledger fantasy
The Burrow is no longer sealed off. More movement means more opportunity, more confusion, and more political noise.

## Festival Ledger rewards focus
- account-wide utility rewards
- early Treasure growth
- early Confidant progression
- mild Strain support items
- one seasonal cosmetic set tied to open roads and patched travel gear

---

## Week 1 — Lucky Draw Week: Fair of Loose Promises

### Week fantasy
A noisy regional fair arrives near Loamwake. Everyone insists it is harmless fun. Nobody can explain why so many useful items seem to appear there.

### Main loop
- earn **Lucky Draws** from Daily Duties, Burrow Posts, Field Returns, and fair-themed Wanderers
- spend them at the **Lucky Wheel**
- convert extras through the **Lucky Stall**
- push scoring tiers for Festival Marks

### Reward tone
This version should feel generous and welcoming.

### Featured reward buckets
- early-use **Treasures**
- **Treasure Shards**
- **Polishes**
- **Mushcaps** bundles
- **Burrowfolk** training items
- one low-rarity cosmetic patchwork clothing line

### Side loop
**Lucky Stall: Patchwork Fair Goods**

Rotating stall examples:
- one reliable growth item
- one event-limited cosmetic
- one beginner-friendly bundle
- one rotating clue fragment purchase

### Story tie-ins
#### Wanderer set
- Peddlers offering suspiciously specific starter bundles
- Oddfolk telling three conflicting stories about who owns the fair

#### Burrow Posts
- **Burrow Post:** “Noise Complaint from Loamwake”
- **Burrow Post:** “Missing Booth Ledger”
- **Confidant Post:** Archivist wants proof that the fair existed last year

### Buried Clue hook
A torn ticket implies the fair’s route matches an old pre-war logistics road.

### Scoring ladder example
- Tier 1: reachable by disciplined early F2P
- Tier 2: reachable by saved pulls + active play
- Tier 3: stretch milestone
- Tier 4: cosmetic / prestige flex tier

## Week 2 — Treasure Week: The Ledgerhollow Appraisal

### Week fantasy
Ledgerhollow opens controlled access to a sealed appraisal archive. Treasure hunters, scribes, and liars all arrive at once.

### Main loop
- earn **Treasure Tickets**
- target specific pools through **Treasure Finder**
- refine desired Treasure families through **Treasure Exchange**
- push ladder rewards while feeding Festival Ledger

### Reward tone
This week should feel more deliberate than Lucky Draw Week. Less excitement, more account planning.

### Featured reward buckets
- targeted Treasure families
- **Treasure Shards**
- **Polishes**
- **Confidence Trail** progression items
- **Lore Map** growth support
- archive-themed cosmetics or interface accents

### Side loop
**Treasure Exchange: Appraisal Desk**

Example rotational targets:
- one Pathwheel-compatible Treasure set
- one Burrow production Treasure set
- one battle-focused Treasure set
- one memory / clue Treasure set

### Story tie-ins
#### Wanderer set
- Messengers delivering contradictory archive permits
- Scrappers trying to offload fake appraisal tags
- Pilgrims insisting one Treasure belongs in Memory Fen

#### Burrow Posts
- **Burrow Post:** “Ledgerhollow Access Appeal”
- **Stranger Post:** “Authenticity Complaint”
- **Confidant Post:** High Confidant requests a private review of one item before public release

### Buried Clue hook
An appraisal mark found on a damaged Treasure matches a sigil later seen in The Buried Court.

### Scoring ladder example
- Tier 1: enough targeted value for all active players
- Tier 2: first “build identity” tier
- Tier 3: account-optimization tier
- Tier 4: high-precision collector tier

## Week 3 — Echo Week: Vigil of the Quiet Fen

### Week fantasy
Memory Fen enters a strange calm. Echo offerings become unusually potent. Smoke Whispers multiply.

### Main loop
- gather **Echoes** and **Echo Rushes**
- present them through the **Ancestral Depictor**
- score progress through **Echo Watch**
- gain **Echo Shards** and Festival Marks

### Reward tone
More ritual, more tension, less carnival. The week should feel slower and more atmospheric.

### Featured reward buckets
- **Echo Shards**
- memory-focused Treasures
- Strain progression support
- Hushline-adjacent clue fragments
- ritual cosmetics, smoke effects, whisper-themed accents

### Side loop
**Echo Watch: Fen Readings**

Example side-loop hooks:
- track total Echoes presented
- watch a weekly “silence meter” shift over time
- unlock extra rewards when community or server thresholds are met

### Story tie-ins
#### Smoke Whispers
- a half-remembered warning about Crownroot record edits
- a vision of a Warden wearing an older title
- a childlike voice asking why the Fen remembers more than the Burrow does

#### Burrow Posts
- **Burrow Post:** “Request for Additional Pipeleaf”
- **Stranger Post:** “Pilgrim Refuses to Leave the Waterline”
- **Wild Post:** The Wild Ones are seen avoiding the Fen entirely

### Buried Clue hook
The first direct mention of the phrase “Memory Shift” appears here.

### Scoring ladder example
- Tier 1: one ritual cycle completed
- Tier 2: efficient Echo saving plus regular activity
- Tier 3: deep hoarding target
- Tier 4: season-defining ritual capstone

# Cycle Example 02 — The Fractured Records Season

## Theme
Public history becomes unstable. Competing versions of the old world spread through Wanderers, Posts, and archive leaks. The season should feel more political and slightly less trustworthy.

## Festival Ledger title
**Festival Ledger: Split Records**

## Festival Ledger rewards focus
- archive and clue rewards
- Crownroot / Hushline tension
- Standing cosmetics
- stronger Confidant progression items

---

## Week 1 — Lucky Draw Week: The Clerkless Fair

### Week fantasy
The fair returns, but no one can identify who filed the permits. Booth inventories keep changing overnight.

### Notable differences from Cycle 01
- more fake reward teasers
- more decoy mini-prizes
- additional Lucky Stall reroll decision
- stronger Burrow Post comedy around missing clerks

### Story tie-ins
- Oddfolk speak in identical phrases despite claiming different origins
- Loose Quills begin appearing as named side characters
- Archivist Confidants dispute whether last season happened as remembered

---

## Week 2 — Treasure Week: Claims of Ownership

### Week fantasy
Ledgerhollow’s appraisal chamber becomes a legal battlefield. Every Treasure now arrives with multiple competing claims.

### Side loop variant
**Treasure Exchange: Claim Desk**

New pressure:
- choose between safer public appraisal rewards or riskier private claim routes
- some targets give better short-term value, others unlock hidden clue sets

### Story tie-ins
- Dissidents attempt to smuggle sealed family claims
- High Confidants push political pressure through Burrow Posts
- one Treasure family is quietly removed from public listings mid-week

---

## Week 3 — Echo Week: The Refusal Vigil

### Week fantasy
The Depictor begins returning incomplete responses. Echoes still register, but certain names produce silence.

### Side loop variant
**Echo Watch: Refusal Readings**

New pressure:
- community mystery track
- one or two hidden thresholds only visible through Buried Clue progress
- more direct link to Deepening progression

### Story tie-ins
- Smoke Whispers now contradict each other
- Witness Confidants begin using different ritual terminology
- a Remembered Warden silhouette appears in the smoke for the first time

# Cycle Example 03 — The Open Dispute Season

## Theme
Tensions between Cliques, Wanderers, and region authorities become more visible. The Burrow is more connected, but also more exposed. This season should feel like the first point where soft social systems begin leaning toward bigger conflict.

## Festival Ledger title
**Festival Ledger: Gathered Claims**

## Festival Ledger rewards focus
- Clique cooperation rewards
- Burrowfolk support materials
- Standing cosmetics
- public-status flex rewards

---

## Week 1 — Lucky Draw Week: The Public Draw

### Week fantasy
Lucky Wheel prizes are displayed openly for the first time. Everyone argues the draw is rigged, then participates anyway.

### Side loop variant
**Lucky Stall: Public Counter**

Special pressure:
- public/server meter tracks total community draws
- side rewards unlocked through combined participation
- additional debate Posts increase event flavor without changing core math

### Story tie-ins
- Messengers deliver public notices from competing Cliques
- Rousers cause light chaos around the stall
- Burrow Posts frame the week as “healthy civic excitement” despite obvious disorder

---

## Week 2 — Treasure Week: The Standing Review

### Week fantasy
Treasures are evaluated not only for power, but for who should be allowed to hold them.

### Side loop variant
**Treasure Exchange: Standing Review**

Special pressure:
- players choose between direct account upgrades and standing-linked side rewards
- Confidant role tags matter more in event flavor text
- Broker and High Confidant story hooks appear more often

### Story tie-ins
- Defectors leak review criteria
n- Scrappers begin selling forged standing seals
- Buried Clues imply the review system existed before the fall

---

## Week 3 — Echo Week: The Public Silence

### Week fantasy
Echo activity becomes a public matter. The community watches which names the Depictor answers and which it refuses.

### Side loop variant
**Echo Watch: Public Silence**

Special pressure:
- weekly meter tied to “answered” versus “refused” offerings
- server-wide curiosity hook
- direct clue connection into the next Deepening chapter

### Story tie-ins
- Smoke Whispers name a faction not yet publicly introduced
- Turned Hands warn players not to trust the week’s public narrative
- one Wanderer class disappears after this cycle and returns later in altered form

# Long-Arc Scaling Notes

## 1. Weekly events should support recurring cast growth
Just like roaming post systems, the weekly events should be able to absorb a large recurring cast over time.

By year 2+, each week type may include:
- dozens of named Wanderer appearances
- rotating Runaway rumors
- repeat Confidant reactions that evolve by Deepening
- variant side-loop operators
- legacy clue callbacks

## 2. Event density should increase by layering, not replacement
A good seasonal expansion adds:
- one new reward family
- one new story hook family
- one new side-loop twist
- one new returning cast subset
- one new hidden-clue branch

It should not require rebuilding the full weekly skeleton.

## 3. Old seasonal content should be remixable
Any later season should be able to remix:
- prior Lucky Stall operators
- prior Treasure Exchange themes
- prior Echo Watch readings
- past Burrow Posts rewritten with new political context
- returning Wanderers now carrying old knowledge

## 4. Deepening should change event interpretation
The week names remain stable.
The meaning changes as the player learns more.

Examples:
- early **Lucky Draw Week** feels playful; later it may feel suspiciously engineered
- early **Treasure Week** feels rewarding; later it may expose ideological control over history
- early **Echo Week** feels mystical; later it may reveal selective memory censorship
