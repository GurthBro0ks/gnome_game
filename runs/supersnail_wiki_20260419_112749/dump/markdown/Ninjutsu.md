1. Ninjutsu
Ninjutsu is the unique mechanic in Yamato and has four types: Fire, Aqua, Thunder, and Earth. Each type corresponds to one of the Snail's HARD stats. By hitting the grids relating to Ninjutsu (named Soul of [type]) durring exploration, you can acquire the necessary points for upgrading each type. 

At first, you can only upgrade each jutsu to Tier 4, but through INTEL you can upgrade it to maximum (Tier 20). In addition to these tiers, each type has 3 special techniques with 5 levels each that can be upgraded for an extra boost. Upgrading each of these techniques to maximum rewards s, which are used to upgrade Yamato's Realm Trait, the . 

Once each jutsu is at its maximum (Tier 20), a challenge from the ninja of that jutsu will appear on the map. Defeating each of the four ninjas will unlock the final, fifth ninja and completing all of these will complete the Yamato Key Clue "Ultimate Ninja Challenge."

1. Stats
You have the option to Practice one of the four Jutsu, which will gain experience for that Jutsu much faster. This experience is used to upgrade each Jutsu to the next tier. The recommended order is Fire > Thunder > Earth > Aqua.
